About
RxManager Pro v101 — Phase 8-F

Quick-reference prescription tool for junior doctors. Personal data — prescriptions, notes, photos — is stored locally on your device. The core drug library is served from Supabase with a local cache for offline use.

v101 — Phase 8-F: Full bulk-fetch strategy activated. USE_BULK_FETCH = true. Cold starts now call get_all_conditions_full() — a single Postgres RPC that returns every condition's prescriptions in one shot (~47 KB). The result is written to IDB rx_cache in a single batch transaction. All subsequent condition opens are zero-network, zero-skeleton until content changes. On warm launches where content is unchanged, the app makes exactly 1 network call (the ~100-byte version check), then renders everything from IDB. Full offline capability after first successful launch. Rollback: set USE_BULK_FETCH = false.
